# 自制天津工业大学本科毕业论文LATEX模板（娱乐非官方）
本模板改自：https://github.com/BHOSC/BUAAthesis ，着重强调这个模板是自己写着玩的，工大没有latex模板，用latex写完学校查重和提交论文也不支持这个格式，word模板在`word模板`文件中。
## 写这个模板的原因
1. 没有学过word排版
2. word的排版太丑相对于latex
3. word公式太难输入
4. 十分想学习latex
5. 无聊
